Comfyui图狗管理器环境管理页面功能介绍

镜像源选择

  • 清华大学源：国内速度快，更新及时
  • 阿里云源：国内速度快，稳定
  • 中国科技大学源：国内速度快，更新及时

  • 国内用户优先选择国内镜像源，下载速度更快

  • 设置会自动保存，下次启动时自动使用上次的选择
功能一：通过requirements.txt安装

📝 功能说明

  • 批量安装requirements.txt文件中列出的所有依赖
  • 自动处理依赖关系
  • 支持版本指定安装

🔍 使用步骤

  1. 点击"选择文件"按钮选择requirements.txt文件
  2. 确认文件路径无误后点击"开始安装"
  3. 等待安装过程完成
  4. 查看输出区域的安装结果和详细信息

⚠️ 注意事项

  • 确保requirements.txt格式正确
  • 安装过程中请勿关闭程序
  • 如有特殊依赖（如.whl文件）需单独处理
功能二：单个依赖安装卸载

📝 功能说明

  • 安装或卸载单个Python包
  • 支持指定版本安装
  • 支持完全卸载包及其依赖

🔍 使用方法

  1. 在输入框中输入包名（可选择指定版本，如：numpy==1.21.0）
  2. 点击"安装"按钮进行安装
  3. 点击"卸载"按钮进行卸载
  4. 等待操作完成并查看结果

💡 使用技巧

  • 安装时可以使用版本号限定：
    • 包名==1.0.0：安装指定版本
    • 包名>=1.0.0：安装大于等于某版本
    • 包名<=1.0.0：安装小于等于某版本
  • 卸载前建议先确认没有其他包依赖它
功能三：安装.whl文件

📝 功能说明

  • 安装本地的.whl格式安装包
  • 支持自动处理依赖关系
  • 适用于离线安装或特定版本安装

🔍 使用步骤

  1. 点击"选择文件"按钮选择.whl文件
  2. 确认文件路径无误后点击"开始安装"
  3. 等待安装完成
  4. 查看安装结果

⚠️ 注意事项

  • 确保.whl文件与Python版本兼容
  • 确保.whl文件与系统架构匹配（32位/64位）
  • 部分包可能需要提前安装依赖
功能四：通过链接安装whl

📝 功能说明

  • 直接通过URL下载并安装.whl文件
  • 支持进度显示和状态提示
  • 自动处理下载和安装过程

🔍 使用步骤

  1. 在输入框中粘贴.whl文件的下载链接
  2. 点击"下载并安装"按钮
  3. 等待下载完成和安装完成
  4. 查看安装结果

⚠️ 注意事项

  • 确保链接是直接指向.whl文件的URL
  • 需要稳定的网络连接
  • 大文件下载可能需要较长时间
功能五：检查依赖冲突

📝 功能说明

  • 检查requirements.txt中的依赖关系
  • 识别版本冲突和缺失的包
  • 提供详细的依赖分析报告

🔍 使用步骤

  1. 点击"选择文件"按钮选择要检查的requirements.txt
  2. 点击"开始检查"按钮
  3. 等待检查完成
  4. 查看检查报告

💡 检查结果说明

  • 缺失的依赖：需要安装的包
  • 冲突的依赖：版本不匹配的包
  • 正确安装的依赖：无需处理的包
功能六：生成requirements.txt

📝 功能说明

  • 导出当前环境中已安装的所有包
  • 自动生成requirements.txt文件
  • 包含版本信息

🔍 使用步骤

  1. 点击"生成当前环境的requirements.txt"按钮
  2. 等待生成完成
  3. 文件会保存在程序所在目录

💡 使用建议

  • 在环境配置完成后生成，方便环境复制
  • 定期备份requirements.txt
  • 可用于项目迁移和环境重建

📌 通用注意事项

  • 操作前确保Python路径配置正确
  • 重要操作前建议备份环境
  • 安装过程中保持网络连接稳定
  • 遇到问题时查看输出区域的错误信息
  • 需要管理员权限时请以管理员身份运行程序

🌟 最佳实践建议

  • 优先使用requirements.txt进行批量安装
  • 定期检查依赖冲突
  • 保持包版本的稳定性
  • 重要更新前先测试兼容性
  • 定期清理不需要的包
